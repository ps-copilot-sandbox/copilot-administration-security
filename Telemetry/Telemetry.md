# Demo: Enterprise Telemetry and Audit Logs

> **Audience:** GitHub Copilot Administrators
> **Goal:** Walk administrators through enterprise Copilot telemetry insights, audit log access, and related Copilot API references.

---

## Setup

Have the participant share their screen and navigate to:

```
https://github.com/enterprises/[ENTERPRISE]/insights/copilot
```

There are two main sections to review: **Copilot usage** and **Code generation**.

---

## Copilot Usage

This section provides an overview of how Copilot is being used across the enterprise.

Walk through:

- All graphs on the insights page
- NDJSON download option at the top-right
- Date filter options for changing the reporting window

---

## Code Generation

This section provides insights into code being generated by Copilot.

Walk through:

- All graphs on the insights page
- NDJSON download option at the top-right
- Date filter options for changing the reporting window

---

## Audit Logs

Demonstrate the two ways to open the enterprise audit log.

### Method 1

- Open the enterprise where you are an admin
- Select **Settings** at the top
- Select **Audit log** in the left navigation

### Method 2

- Open the enterprise where you are an admin
- Select **AI Controls** at the top
- Select **Agents** on the left (if not selected automatically)
- Scroll down and select **Audit logs**
- This routes to the same audit log view as Method 1

## Copilot API Reference Walkthrough

Walk through relevant Copilot REST API calls step-by-step against an example organization with Copilot enabled, and explain:

- The purpose of each endpoint
- The returned data and how to interpret it

Reference:

- https://docs.github.com/en/rest/copilot?apiVersion=2026-03-10

## Key Takeaways

- Enterprise telemetry provides visibility into Copilot usage trends and code generation activity.
- NDJSON exports support deeper analysis and operational reporting.
- Audit logs and API data together improve governance, traceability, and administrator oversight.
